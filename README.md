# Gesture Racing Car · 手势控制赛车

基于 Three.js + MediaPipe 的手势控制赛车游戏。通过手部动作控制方向和速度。

## ✨ 功能

- ✋ **手势转向**：左手位置控制方向
- ✊ **握拳加速**：右手握拳控制速度
- 🏎️ **3D 赛道**：Three.js 渲染沉浸式场景

## 🛠 技术栈

HTML5 · CSS3 · JavaScript · Three.js · MediaPipe

## 🚀 快速开始

```bash
git clone https://github.com/chenqiyue123/gesture-racing-car.git
cd gesture-racing-car
# 用 Live Server 打开 index.html
# 允许摄像头权限
```
